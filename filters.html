filters module¶

Defines a few helpful functions for constructing filters on BibTeX entry. For instance, different filters can be conjoined into a predicate so boilerplate is reduced.

filters.get_conjunction_filter(*fns)[source]¶
filters.get_conjunction_filter(*functions)[source]
Parameters:fns (list) – List of predicates
Returns:A unary function rejecting entries whose which do not satisfy all given predicates.
Return type:function

filters.get_mytype_filter(name)[source]
Parameters:mytype (string) – Name to filter for
Returns:A unary function rejecting entries whose mytype field does not equal the given value. The match is case-insensitive.
Return type:function

filters.get_person_filter(name)[source]
