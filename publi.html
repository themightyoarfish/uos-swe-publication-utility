save()[source]¶
save()[source]

Serialize self to pickled file named db.pckl for caching and update the database_file with all entries. Also, directories abstract, pdf and bib are created under prefix which are filled with one file per key each – one for a BibTeX file with the single entry, one text file for the abstract (if present in the entry) and one pdf file (if a pdf named like the entry’s key was found in pdf_dir)

publi.generate_key_swe(entry)[source]¶
publi.generate_key_swe(entry)[source]

Create a new bibtex key for the given entry in the format desired by the group leader. The format is <first author>:<first title word>:yy. Stop words (non-significant ones such as articles) are ignored. Missing fields are empty.

Parameters:entry (pybtex.database.Entry) – Bibliography item to key
Returns:The new key
Return type:str
